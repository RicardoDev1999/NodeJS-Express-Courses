# NodeJS-Express-Courses


The pourpose of this project is to learn how to build modern JS projects. 

Pattern > 
- MVC

Stack > 

- NODE.JS
- Express.JS (Framework)
- EJS (View Engine)
- Joi (Schema Validator)
- MongoDB (Database, using Atlas as the server)
